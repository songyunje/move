프로젝트명 : 나만의 패션 옷장
개발목적 : 가지고 있는 옷들을 쉽게 확인하고 오늘의 날씨를 참고하여 오늘의 패션을 정해보자
개발인원 : 4명
개발 기간 : 4.3~5.15
주요 기능 : - crud기능
개발 언어 : java,spring,vue.js 
개발 환경 : Eclipse, tomcat,VScode
DB : MySQL


기능 => 로그인, 로그아웃 
       => 내 옷장(글 작성, 삭제,수정)

       => 오늘의 날씨( 네이버 날씨 등에서 가져오기)
       => 내 패션 조합(글 작성, 삭제)-> 내 옷장의 옷들로 링크
       => 위시리스트 (패션 사이트 링크 넣으면서 작성)
